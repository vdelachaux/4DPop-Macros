## Description

`4DPop_TEST_Macros` is a developer sandbox. When the test method does not yet exist, it prompts to create a starter `4DPop_TEST_Macros` method (setting its attributes and opening it in the code editor). Otherwise it runs the sandbox code — for example converting `New object(...)` calls to object-literal syntax, evaluating the highlighted expression, or parsing declarations to display the DECLARATION dialog.
